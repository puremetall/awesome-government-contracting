# Contributing

Thanks for helping improve **Awesome Government Contracting**! This list aims to be
the most useful, trustworthy directory of government contracting resources on GitHub.

## How to contribute

1. Fork the repository and create a branch.
2. Add your resource to the most relevant section in `README.md`.
3. Keep the format consistent: `- [Name](url) — short, neutral description.`
4. Open a pull request describing what you added and why it belongs.

## Guidelines

- **Be useful.** Add resources that genuinely help people find or win government contracts.
- **Prefer free / open resources.** Paid tools are allowed if they offer clear, unique value; mark them as paid.
- **Stay neutral.** Descriptions should be factual, not promotional.
- **No dead links.** Verify URLs before submitting.
- **One resource per line**, alphabetized within a section where practical.
- **State & local additions are especially welcome** — help us cover more states, counties, and cities.

## Adding a new section

If a resource doesn't fit an existing section, propose a new one in your PR. Keep the
table of contents in sync.

## Code of conduct

Be respectful and constructive. We review PRs in good faith and appreciate every contribution.
